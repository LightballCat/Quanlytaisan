	Project -> Manage NuGet Packages... -> Browse 
	tìm kiếm "Microsoft.ReportingServices.ReportViewerControl.Winforms" -> Install

	Extentions -> Manage Extentions -> Tìm kiếm "Microsoft RDLC Report Designer" -> Install
